# Scenario Certification

This directory holds executable scenario certification assets once the scenario
engine is available. The initial required scenario set is:

- `BA-001`: AIS vessel appears, moves, disappears.
- `BA-007`: UI receives live stream update.
- `ADS-002`: operator creates investigate task.
- `ADS-003`: asset accepts task.
- `ADS-004`: asset reports en route, executing, complete.

The contract-level expectations are pinned in `cert/lattice/assertions.yaml`.
