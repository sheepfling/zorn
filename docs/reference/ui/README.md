# Zorn UI Reference Registry

This directory preserves the UI research artifacts used to shape Zorn's COP/debug
UI. These files are reference material, not vendored application code.

- `zorn_lattice_ui_reference_registry.md` is the detailed source, image, and
  requirement registry.
- `zorn_lattice_ui_reference_registry.json` is the machine-readable registry
  companion.
- `zorn_lattice_ui_resource_registry.md` is the concise resource and feature
  taxonomy.

The old Stratum scaffold zip from the intake bundle is intentionally not unpacked
here. Zorn should absorb the requirements and build its own UI/plugin surface
against the current Zorn API and certification contracts.
